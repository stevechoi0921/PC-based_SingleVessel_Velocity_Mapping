# PC-based Single Vessel Velocity Mapping

MATLAB code and example data for phase-contrast based single-vessel velocity estimation.

## Files
- `Figure2_Velocity_plot.m` – Main plotting script
- `flow_data_Ar.mat` – Arterial flow data
- `flow_data_Ve.mat` – Venous flow data
- `S2_Data.xlsx` – Supplementary data

## Requirements
- MATLAB R2021a or later
